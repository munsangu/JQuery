show(), hide(), toggle()

선택한 요소를 나타나게 하는 것(show)과 순간적으로 사라지게 하는 것(hide)

이 단락이 ON / OFF

fadeIn(), fadeOut(), fadeToggle(), fadeTo()

CSS opacity 변경

slideUp(), slideDown(), slideToggle()

delay()

stop(), finish()

stop: 선택한 요소에서 실행 중인 모든 이펙트 효과를 즉시 중시

finish: 선택한 요소가 포함된 큐까지 제거하여 모든 이펙트 효과를 종료시킴

jQuery.fx

이펙트 효과가 구현되는 방법을 제어

  1. jQuery.fx.speeds: 밀리초에 해당하는 slow(600), normal(400), fast(200) 등의 값을 가지고 이펙트 효과의 속도를 나타냄
  2. jQuery.fx.interval: 이펙트 효과가 보여지는 동안의 초당 프레임 수를 나타냄
  3. jQuery.fx.off: 모든 이펙트 효과를 사용할 수 없도록 비활성화시킴

animate()

$(선택자).animate(프로퍼티,[지속시간],[시간당속도함수],[콜백함수])

  1. 프로퍼티: 필수값, 이펙트 효과를 구성할 CSS 속성을 정의
  2. 지속시간: 이펙트 효과가 지속될 시간을 전달
  3. 시간당속도함수(easing function): 이펙트 효과의 시간당 속도를 전달
  4. 콜백 함수: 이펙트 효과의 동작이 완료된 후에 실행할 함수

이펙트효과가 모두 끝나면 변경될 텍스트

animate() - 미리 정의된 값 사용: slow, hide, toggle
